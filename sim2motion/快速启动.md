# 快速开始指南 - Sim2Sim PM01 Data Recorder
默认输出fps是100Hz，数据保存为CSV和PKL格式。

## 文件说明

- `sim2sim_pm01.py` - 主程序脚本（带 GUI 的数据录制工具）
- `test_dependencies.py` - 依赖测试脚本
- `README_sim2sim.md` - 详细使用说明文档

## 快速开始

### 1. 测试依赖

首先运行测试脚本，确保所有依赖都已正确安装：

```bash
cd /home/abo/git/zqsa01_legged_gym/legged_gym/sim2motion
python test_dependencies.py
```

如果看到 `✓ All dependencies are installed!`，说明所有依赖都已就绪。

### 2. 运行主程序
进入zqsa01_legged_gym的conda虚拟环境，否则没有legged_gym包，运行会报错
```
conda activate zqsa01
```

```bash
python sim2sim_pm01.py
```

### 3. 使用 GUI

运行后会显示一个 GUI 窗口：

- **Start Recording** - 开始录制数据
- **Stop Recording** - 停止录制并保存
- **Status** - 显示当前状态
- **Frames** - 显示已录制的帧数

### 4. 键盘控制

在仿真运行时，可以使用键盘控制机器人：

| 按键 | 功能 |
|------|------|
| 2 | 前进 |
| 3 | 后退 |
| 4 | 向左 |
| 5 | 向右 |
| 6 | 逆时针旋转 |
| 7 | 顺时针旋转 |
| 0 | 重置速度 |

### 5. 数据保存

点击 "Stop Recording" 后，数据会自动保存到当前目录：

- `pm01_motion_YYYYMMDD_HHMMSS.pkl`
- `pm01_motion_YYYYMMDD_HHMMSS.csv`

## 典型工作流程

```
1. python sim2sim_pm01.py
2. 等待 GUI 窗口和仿真加载完成
3. 使用键盘控制机器人到合适位置
4. 点击 "Start Recording"
5. 控制机器人执行动作（行走、站立等）
6. 点击 "Stop Recording"
7. 数据自动保存到当前目录
```

## 数据格式

### CSV 格式（31列）

```
列 0-2:  root_pos (x, y, z)        # 根节点位置
列 3-6:  root_rot (w, x, y, z)     # 根节点姿态（四元数）
列 7-30: dof_pos (24个关节角度)    # 关节角度
```

### PKL 格式

```python
{
    "fps": 100,                           # 帧率
    "root_pos": np.ndarray(N, 3),        # 根节点位置
    "root_rot": np.ndarray(N, 4),         # 根节点姿态
    "dof_pos": np.ndarray(N, 24),         # 关节角度
    "local_body_pos": np.ndarray(N, 29, 3), # 身体部位位置
    "link_body_list": list,               # 身体部位名称
}
```

## 故障排除


## 进阶配置

### 修改仿真参数

在 `Sim2simCfg` 类中修改：

```python
class Sim2simCfg(PM01Cfg):
    class sim_config:
        mujoco_model_path = '...'  # 模型路径
        sim_duration = 1000.0      # 仿真时长
        dt = 0.001                # 仿真步长
        decimation = 10           # 降采样倍数
```

### 修改录制频率

录制频率 = 1 / (dt * decimation)

例如：
- dt = 0.001, decimation = 10 → 100 Hz
- dt = 0.001, decimation = 20 → 50 Hz

## 示例代码
